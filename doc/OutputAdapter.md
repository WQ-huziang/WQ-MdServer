# OutputAdapter

1. Setter

- Setfilepath (设置文件路径)
- SetUdpHtons (设置udp端口)
- SetTcpHtons (设置tcp端口)
- SetUdpAddr (设置udp地址)
- SetTcpAddr (设置tcp地址)

2. Getter

- GetUdpSocket (获取udp连接的socket)
- GetTcpSocket (获取tcp连接的socket)
- GetFilePath (获取文件路径)

3. Adapter

- UdpOutputAdapter (udp连接适配器)
- TcpOutputAdapter (tcp连接适配器)
